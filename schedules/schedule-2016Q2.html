Program ID codes: L=Lowell, B=BU, N=NAU, M=UMd/GSFC, T=UToledo, E=Engineering

Dark code (D=dark, g=gray, b=bright) methodology: At a time resolution of 1 minute, calculate the fractional illumination of the moon for the allocated time (ignoring time during astronomical/nautical/civil twilights & setting to zero for when the moon is below the horizon). Determine the dark code from the mean value of this metric where D=dark for <=0.25, g=gray for <=0.7, and b=bright for >0.7.

Remote: If entry is "Remote Requested", then remote observing request is pending and not yet approved. If entry is "Remote" then remote request is approved. If entry is blank, then observers are expected to be on-site at DCT.

Bolded red items are Time Critical Observations, during which ToO interrupts will not be accepted.

Italicized gray items are engineering optional interrupts (EOI) for brief testing. Principal will be notified before the start of the prior night if any of this time will be used for a quick engineering test.
