Program ID codes: L=Lowell, B=BU, G=UT/KASI(IGRINS), M=UMd/GSFC, N=NAU, NL=Joint NAU/Lowell, T=UToledo, Y=Yale, E=Engineering

Dark code (D=dark, g=gray, b=bright) methodology: At a time resolution of 1 minute, calculate the fractional illumination of the moon for the allocated time (ignoring time during astronomical/nautical/civil twilights & setting to zero for when the moon is below the horizon). Determine the dark code from the mean value of this metric where D=dark for <={0}, g=gray for <={1}, and b=bright for >0.7.

Remote: If entry is "Remote Requested", then remote observing request is pending and not yet approved. If entry is "Remote" then remote request is approved. If entry is blank, then observers are expected to be on-site at DCT.
